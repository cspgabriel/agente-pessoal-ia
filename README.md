# 🤖 Agente Pessoal de IA

Aprenda a criar seus próprios agentes de IA e automatizar tarefas repetitivas — mesmo sem saber programar.

**Kit completo e gratuito.** Cole no ChatGPT/Claude/Gemini e comece em 5 minutos.

---

## 🎯 O Que É

O **Agente Pessoal de IA** é uma célula de **3 mentores de IA** que te ensinam a:

- **Criar agentes** personalizados para qualquer tarefa
- **Automatizar** tarefas repetitivas (pesquisa, escrita, análise, organização)
- **Orquestrar** múltiplos agentes em pipeline
- **Recuperar horas** por semana delegando trabalho para IA

## 🚀 Como Usar (5 minutos)

### Caminho rápido (5 min)
1. Abra o arquivo **PROMPT-MESTRE.md**
2. Copie o bloco entre `---INÍCIO---` e `---FIM---`
3. Cole no ChatGPT (versão grátis funciona)
4. Responda as 6 perguntas
5. Receba seu agente personalizado

### Caminho completo (15-20 min)
1. Abra o arquivo **AGENTE-PESSOAL-IA.md**
2. Cole todo o conteúdo no ChatGPT, Claude ou Gemini
3. O Mestre Mentor vai conduzir você pelas 3 fases
4. Use o **BONUS-01** para aprender a criar novos agentes
5. Consulte o **BONUS-02** para automatizar tarefas
6. Teste os **50 prompts prontos** do BONUS-03
